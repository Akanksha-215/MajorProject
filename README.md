# Computer Vision Project

A full-stack application for computer vision processing with React frontend and Python backend.

## Project Structure

```
├── frontend/          # React application
├── backend/           # Python Flask API
├── README.md         # Project documentation
└── requirements.txt   # Python dependencies
```

## Features

- **File Upload**: Support for both image and video files
- **Multiple Detection Parameters**:
  - Lane Detection
  - Vehicle Detection
  - Pedestrian Detection
  - Traffic Sign Recognition
  - Object Tracking
- **Real-time Processing**: Backend processing with OpenCV
- **Modern UI**: Clean React interface with loading animations

## Setup Instructions

### Backend Setup
1. Navigate to backend directory
2. Install Python dependencies: `pip install -r requirements.txt`
3. Run the Flask server: `python app.py`

### Frontend Setup
1. Navigate to frontend directory
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

## Usage

1. Upload an image or video file
2. Select the detection parameter(s)
3. Click "Generate" to process
4. View the results with processed output
